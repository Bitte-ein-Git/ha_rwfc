## Installation (Manual)
1. Add this Repository to HACS:
   - HACS > 3 dots > "Add custom repository"
   - URL: `Bitte-ein-Git/ha_rwfc`
   - Type: Integration

2. Select "🏎️ RWFC Status tracker".
